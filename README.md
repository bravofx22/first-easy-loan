# First Easy Loan

A **dark-themed professional loan website** using HTML, CSS, and Firebase Authentication.  
Hosted on **GitHub Pages**. Now fully upgraded with **interactive Jobs & Surveys tabs**.

---

## Features

- **Register & Login Users:** Secure email/password authentication using Firebase  
- **Dashboard with Tabs:**  
  - **Home:** Loan calculator  
  - **Jobs:** Apply for jobs and view job list  
  - **Surveys:** Submit survey responses  
  - **Profile:** View email, number of jobs applied, surveys submitted, and logout  
- **Responsive Dark Theme:** Works perfectly on mobile & desktop  
- **Firestore Integration:** Save job applications & survey submissions for each user  
- **Professional UI:** Cards, buttons, hover effects

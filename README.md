# First Easy Loan

A **dark-themed loan website** using HTML, CSS, and Firebase Authentication.  
Hosted on GitHub Pages.

## Features
- Register & Login users
- Dashboard with Loan Calculator
- Responsive dark theme
- Firebase authentication (email/password)

## Setup
1. Replace Firebase placeholders in `index.html`:
   - apiKey
   - messagingSenderId
   - appId
2. Commit & push to GitHub
3. Open: https://bravofx22.github.io/first-easy-loan/
